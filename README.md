## Getting Started

> Project in development (not deployed yet on npm).

### Prerequisites

- Node 16 or higher
  1. Install NodeJS from [NodeJS Official Page](https://nodejs.org/en/download/)
  2. Open Terminal
  3. Check Current NodeJS Version
  ```sh
  node -v
  ```
